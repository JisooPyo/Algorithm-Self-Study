## 너비 우선 탐색(BFS, Breadth-First Search)

최대한 넓게 이동한 다음, 더 이상 갈 수 없을 때 아래로 이동

### 예시

* 친구 관계를 그래프로 표현했을 때, Sam과 Eddie 사이에 존재하는 경로를 찾는 경우
* 위치들을 그래프로 표현했을 때, 각 위치간의 최단 거리를 찾는 경우
